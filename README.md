#  Word Frequency & Text Statistics Analyzer
This is a Python-based command-line tool for text analysis.
A command-line tool that analyzes any plain-text
file to produce detailed word-frequency, vocabulary, and readability statistics. Built entirely with core Python (dictionaries, string processing, and file I/O), it
demonstrates the foundational counting and text-normalization techniques that
underlie modern NLP pipelines like TF-IDF and search indexing.
